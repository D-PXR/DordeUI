# DordeUI
A WoW UI created using Ellesmere UI, Chattynator, and Details!

The goal of this UI was to emulate a cooldown manager without actually using most of the cooldown manager provided by blizzard due to how little customization there is. This is my current solution, a clean, focused UI that has minimal reliance on addons. This solution may change depending on my needs. I will try and keep this UI updated as and when I make changes. A Changelog can be found at the bottom of this ReadMe document.

DordeUI is a 0.71 @ 1920x1080 scale setup, other resolutions have not been tested. This means that if you have a higher resolution monitor, the import process will be less "plug and play". 

### **Addons Used:** 
- EllesmereUI
- Chattynator
- Details!

### **Setup:**    
1. Import the EllesmereUI profile.
2. Import the Details! profile.
3. Set up chattynator using the settings provided in "Chattynator Settings"
